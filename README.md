# Aniket Portfolio

Personal portfolio site built with React + TypeScript + Vite + Tailwind CSS.

## Project Structure

```text
portfolio/
├── index.html                 # HTML entry template
├── package.json               # Scripts + dependencies
├── package-lock.json          # Locked npm versions
├── vite.config.ts             # Vite config + plugins + path aliases
├── tsconfig.json              # TypeScript config
├── src/
│   ├── main.tsx               # React bootstrap
│   ├── App.tsx                # Section layout
│   ├── index.css              # Global styles + theme tokens
│   ├── data/
│   │   └── portfolio.ts       # Main content (name, projects, skills, links)
│   ├── components/            # UI sections (Hero, About, Projects, etc.)
│   └── utils/
│       └── cn.ts              # Tailwind class merge helper
├── public/
│   └── images/
│       ├── featured/          # Optional featured project thumbnails
│       └── side-projects/     # Optional side project thumbnails
├── dist/                      # Production output (generated by build)
└── node_modules/              # Installed dependencies (generated by npm install)
```

## Prerequisites

- Node.js `^20.19.0` or `>=22.12.0` (required by Vite 7)
- npm (comes with Node.js)

Check installed versions:

```bash
node -v
npm -v
```

## Install Dependencies

From the project root:

```bash
npm install
```

## Run on Localhost (Development)

```bash
npm run dev
```

Vite will print a local URL (usually `http://localhost:5173/`).

If you want a fixed host/port:

```bash
npm run dev -- --host 127.0.0.1 --port 5173
```

## Build for Production

```bash
npm run build
```

This generates the production output in `dist/`.

Note: this project uses `vite-plugin-singlefile`, so JS/CSS are inlined into `dist/index.html`.

## Preview Production Build Locally

```bash
npm run preview
```

By default preview runs on `http://localhost:4173/`.

Fixed host/port example:

```bash
npm run preview -- --host 127.0.0.1 --port 4173
```

## Where to Edit Portfolio Content

- Main data/content: `src/data/portfolio.ts`
- Section components/layout: `src/components/*.tsx`
- Global styles/theme: `src/index.css`

## Images

Put image files in:

- `public/images/featured/`
- `public/images/side-projects/`

Then reference them in `src/data/portfolio.ts` with paths like:

```ts
image: "/images/featured/my-project.png"
```
